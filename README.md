# UpscalePruebaTecnica
# 🚀 Prueba Técnica - Programador Web (Upscale)

### Candidato: Beckhan Pacheco
**Fecha de entrega:** 22 de marzo de 2026

Este repositorio contiene la solución a la prueba técnica para el puesto de Programador Web. Se ha desarrollado una plataforma de autenticación y gestión de perfil siguiendo los lineamientos de diseño proporcionados en Figma.

## 🛠️ Tecnologías Utilizadas
* **Framework:** ASP.NET Core MVC
* **Base de Datos:** SQL Server
* **Frontend:** Bootstrap 5, CSS3 personalizado y JavaScript (Vanilla)
* **Iconografía:** Font Awesome 6

## 📋 Funcionalidades Destacadas
1. **Autenticación Real:** Validación de credenciales consultando la base de datos SQL Server.
2. **Control de Inactividad:** Implementación de un monitor de actividad que despliega un modal de advertencia con una barra de progreso animada en CSS.
3. **Manejo de Errores:** Mensajes de validación visuales en el formulario de login y lógica de intentos fallidos.
4. **Diseño Pixel Perfect:** Ajuste de estilos CSS para coincidir con la propuesta estética de Figma (sombras, bordes destacados y paleta de colores).

## 🗄️ Configuración de la Base de Datos
Para recrear el entorno de datos:
1. Ejecutar el script ubicado en la raíz: `script_database.sql`.
2. El script creará la base de datos `UpscaleDB` y las tablas de catálogos necesarias.
